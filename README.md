# MathHub

Design assets and page references for the MathHub website.

## Asset structure

- `assets/images/`: exported UI and illustration assets with cleaned English file names.
- `assets/references/screenshots/`: page design references for the planned MathHub screens.

## Screen references

- `main-home.png`: main landing page
- `concepts-overview.png`: concept curriculum overview
- `concepts-detail.png`: concept lesson detail
- `past-exams.png`: past exam archive
- `workbooks-solutions.png`: workbook and solution list
- `usa-resources.png`: USA resources page
